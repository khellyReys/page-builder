WhataHotel Netlify Agent Rules

REQUIRED: Use `createPromo()` from `src/lib/promoFactory.ts` for ALL promos. Read `docs/whatahotel-agent-new-promo.md` first.

Scope: ONLY edit `src/data/promo-N.ts` + `src/data/promos.ts`. Never edit types, components, or factory. Never copy from existing promo files.

```ts
import { createPromo } from "../lib/promoFactory";
export const promoN = createPromo({
  id: "promo-N",
  title: "...",
  dates: "...",
  client: "",
  cityImageUrl: "...",    // optional
  cityImageAlt: "...",    // optional
  hotels: [{
    name: "Hotel Name",
    location: "City, State, Country",
    heroImageUrl: "https://whatahotel.com/content/hotels/ID/file.jpg",
    heroAlt: "Hotel description",
    rooms: [{
      name: "Room Name",
      subtitle: "View · Bed Type",
      badgeText: "Exclusive Rate",
      adr: "$458.63",
      grandTotal: "$2,253.79",
      nights: 4,
      checkIn: "2026-04-05",
      checkOut: "2026-04-09",
      bookUrl: "https://www.whatahotel.com/booking/booking_info.cfm?room=CODE&rate=RATE&hotel=ID&checkin=YYYY-MM-DD&checkout=YYYY-MM-DD&guests=2&children=0&rooms=1",
      images: [{ src: "https://d2573qu6qrjt8c.cloudfront.net/HASH/HASH.JPEG", alt: "Room" }],
      roomHighlights: ["Ocean view with king bed", "Accommodates 2 guests"],
      perks: ["Room Upgrade...", "Breakfast x 2 Daily", "$100 Credit", "Free WiFi", "Late Check-Out"]
    }]
  }]
});
```

Fetch: Each booking URL ONCE, full HTML.

Images: Hero = `ul#subSlides` bg-url, prepend `https://whatahotel.com`. Rooms = `.bookingItem` `ul.booking-img-list` `<a href>` only (NOT `<img src>`), max 2/room. CDNs: `d2573qu6qrjt8c` or `d321ocj5nbe62c`. Skip paceholder -> empty array.

Rates: `adr` = price with currency, no "/night". `grandTotal` = tax-inclusive total from booking page. Use lowest offer rate for `bookUrl`.

badgeText: Extract from marketing line between "WhataHotel!" and "More Info". Fallback: `"Exclusive Rate"`. Never `"Hotel Option X"`.

Multi-hotel: 2+ URLs with different hotelIDs -> separate HotelInput entries. Data isolation per URL.

Defaults: 3 lowest rooms; next id from `promos.ts`; `client: ""` if missing.

The factory handles ALL formatting, labels, pricing alignment, and layout. Agent provides only raw data.

Gate: `npm run build` must pass.
